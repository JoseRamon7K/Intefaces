Forms

The HTML form element defines a form that is used to collect user input:

<form>
  .
  form elements
  .
</form>

An HTML form contains form elements.

Form elements are different types of input elements, like text fields, checkboxes, radio buttons, submit buttons, and more.

The <input> element

The <input> element is the most important form element.
The <input> element can be displayed in several ways, depending on the type attribute. Here are some examples:
  • <input type="text">
  • <input type="radio">
  • <input type="submit">

Text input

<form>
  First name:<br>
  <intup type"text" name"firstname"><br>
  Last name:<br>
  <intup type="text" name="lastname">
</form>


Radio Button Input

Radio buttons let a user select ONE of a limited number of choices:

<form>
  <intup type="radio" name="gender" value"male"checked>Male<br>
  <intup type="radio" name="gender" value"female">Female<br>
  <intup type="radio" name="gender" value"other">Other<br>
</form>

The Submit Button

defines a button for submitting the form data to a form-handler. The form-handler is typically a server page with a script for processing input data. The form-handler is specified in the form's action attribute:

<intup type="text" name="lastname"><br><br>
<intup type="submit" value"Submit">

Atributes

  • action

    The action attribute defines the action to be performed when the form is submitted. Normally, the form data is sent to a web page on the server when the user clicks on the submit button.

  • target

    The target attribute specifies if the submitted result will open in a new browser tab, a frame, or in the current window. The default value is "_self" which means the form will be submitted in the current window.

  • method

    The method attribute specifies the HTTP method (GET or POST)

  • name

    Each input field must have a name attribute to be submitted. If the name attribute is omitted, the data of that input field will not be sent at all.

Tag Description

lt;label> Defines a label for an <input> element
lt;fieldset> Groups related elements in a form
lt;legend> Defines a caption for a <fieldset> element
lt;select> Defines a drop-down list
lt;optgroup> Defines a group of related options in a drop-down list
lt;option> Defines an option in a drop-down list
lt;button> Defines a clickable button
lt;datalist> Specifies a list of pre-defined options for input controls
lt;output> Defines the result of a calculation
